# Консоль сервера
Консоль сервера поддерживает следующие комманды:
- `?` - выводи список всех поддерживаемых комманд. Если написать `? <название комманды>`, то выведет названия аргументов, которые принимает комманда.
- `stop`
- `sql`
- `clients`
- `disconnect`
- `send`
- `request`
- `code`
- `cls` - отчищает экран
